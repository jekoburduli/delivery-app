# Restaurant App 🍕🍔🍣

A simple food ordering app built with HTML, CSS, and JavaScript.  
Features:

- Browse restaurants by location
- Geolocation detection (OpenStreetMap API)
- Login/Signup system
- Add meals to cart and checkout
- Modal-based UI
